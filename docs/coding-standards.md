# Coding Standards

## PHP (PSR-2)

The skeleton comes with Slevomat Coding Standards.

To run the Standards test.

```
$ composer silverstripe-standards
```

To fix your Standards violations you can run.

```
$ composer phpcs-fix
# or
$ composer phpcbf
```
